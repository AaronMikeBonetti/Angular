Pipe Lesson

Pipes are a type of decorator and are used to transform data, takes an input and transforms the data without mutating the original property.

Reverse Pipe

The reverse pipe takes a string and reverses the order of the characters.

This is a reverse string ('hello') using a custom pipe: {{ "hello" | reverse }}

AsyncPipe

This is an impure pipe that allows us to subscribe and unsubscribe to an observable (See observable page here). Its syntactic sugar that also helps with avoiding memory leaks.

Chaining Pipes

Its possible to chain, concat or combine pipes for almost endless data transformations.

You can chain pipes (reverse + uppercase) with 'goodbye': {{ "goodbye" | reverse | uppercase }}

Custom Pipes

You can also create your own pipes to create even further powerful data manipulation!
